This project presents an interactive **Power BI dashboard** for **Blinkit (India's Last Minute App)**, providing insights into sales performance, outlet distribution, item ratings, and product-level analytics. The dashboard allows users to explore various KPIs and trends using multiple filters and visualization panels.
